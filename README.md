# BattouGames website

Static portfolio and support website for BattouGames, hosted with GitHub Pages at:

`https://battougames.com`

## Structure

- `index.html` - studio portfolio and project overview
- `privacy.html` - Tunnelwave privacy policy
- `support.html` - Tunnelwave support information
- `styles.css` - shared responsive styles
- `assets/` - BattouGames hero artwork and Tunnelwave imagery
- `app-ads.txt` - AdMob seller declaration
- `CNAME` - GitHub Pages custom domain

## Publishing

GitHub Pages deploys the `main` branch. Keep `app-ads.txt` at the domain root and do not remove the `CNAME` file.
